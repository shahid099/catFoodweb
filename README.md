# 🐾 WhiskerFeast Co. - Premium Cat Food E-Commerce

Welcome to **WhiskerFeast**, a high-performance, mobile-first e-commerce frontend dedicated to premium feline nutrition. This platform is built using **Next.js** and **Tailwind CSS**, designed to provide cat parents with a seamless, blazing-fast shopping experience from kittenhood to the senior years.

---

## ✨ Features Implemented

* 📱 **Mobile-First Responsive Layouts:** A fully responsive user experience scaling smoothly from mobile screens up to wide computer desktops.
* 🗺️ **Stateful Responsive Navbar:** Includes an integrated dynamic search bar, logo tracking, and a smart mobile hamburger drawer managed via React state.
* ⚡ **Optimized Asset Delivery:** Leveraging the native Next.js `<Image />` component with static imports for layout stability (preventing Cumulative Layout Shift) and auto-generated blur-up placeholders.
* 🛍️ **Dynamic Product Engine:** Interactive client-side product grids showing real-time price formatting, tag badges ("Trending", "Sale"), and reactive "Add to Cart" functionality.
* 🤎 **Thematic Pet Styling:** Built with a custom, premium color palette designed specifically to evoke trust and premium brand identity for pet care compliance.
* 💬 **Floating Help System:** An active floating WhatsApp webhook module pinned out of view bounds with animated desktop expansion capabilities.

---

## 🛠️ Tech Stack

* **Framework:** Next.js (App Router)
* **Library:** React 19 / 20
* **Styling:** Tailwind CSS
* **Asset Optimization:** Next.js Image Optimization Engine

## 🚀 Getting Started

Follow these steps to set up the repository locally on your machine.

### Prerequisites

Ensure you have **Node.js** (v18.x or higher recommended) and **npm** installed on your computer.

### Installation

1. Clone the repository down to your computer:
   ```bash
   git clone [https://github.com/your-username/cat-food-website.git](https://github.com/your-username/cat-food-website.git)
   ```


## Navigate directly into the project directory:
   ```bash
   cd cat-food-website
   ```

## Install the project's node dependencies:
   ```bash
   npm install
   ```

## Running the Development Environment
### Launch the local development environment server:
   ```bash
   npm run dev
   ```
